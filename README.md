# Parkoo – Smart Parking Slot Detection & Reservation System

## Overview
Parkoo is an Android application that allows users to locate and reserve parking slots using Google Maps integration and availability tracking.

## Features
- Location-based parking using Maps API
- Slot availability tracking
- Parking reservation system
- Simple and intuitive UI

## Tech Stack
- Kotlin
- Android SDK
- Google Maps API
- Firebase

## Challenges Faced
- Handling simultaneous booking requests
- Preventing double booking of slots
- Managing consistent slot availability

## Key Logic
The system ensures that a parking slot cannot be double-booked by validating slot availability before confirming reservation.
